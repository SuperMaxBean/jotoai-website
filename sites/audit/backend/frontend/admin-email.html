需在 Resend 中验证 jotoai.com 域名后方可使用

📮 Resend 配置步骤

  1. 在 resend.com 注册账号
  2. 在 Domains 页面添加并验证 jotoai.com 域名（添加 DNS 记录）
  3. 在 API Keys 页面创建一个 API Key
  4. 将 API Key 粘贴到上方输入框并保存
说明：所有站点（闪阅、知识中台、审计、安全护栏、Fasium）的表单提交通知邮件，统一从此处配置发送，无需各站点单独设置。
